关于 Osdlyrics 网易来源启用翻译

Poly000 2021-07-25 in Linux · Views

引言

今天偶然得知 GNU/Linux 下的 Osdlyrics 可用于桌面歌词

但是它暂不支持使用翻译。所以这次说一下怎么修改 Osdlyrics 的网易来源脚本尝试使用翻译歌词。

原理

根据 NeteaseCloudMusicApi ， 我发现网易的翻译LRC和原LRC是区分的。

因此，只要修改 /usr/lib/osdlyrics/lyricsources/netease/ 下的 netease.py：

@@ -86,7 +86,9 @@ class NeteaseSource(BaseLyricSourcePlugin):
         # Avoid processing results with no lyrics.
         if 'nolyric' in parsed or 'uncollected' in parsed:
             raise ValueError('This item has no lyrics.')
-        lyric = parsed['lrc']['lyric']
+        lyric = parsed['tlyric']['lyric']
+        if not lyric:
+            lyric = parsed['lrc']['lyric'] # If there is no translation, use the original lyric instead
         return lyric.encode('utf-8')

这样修改就可以尝试使用翻译歌词。
